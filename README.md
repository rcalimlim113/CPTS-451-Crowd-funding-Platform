# CPTS-451-Crowd-funding-Platform

## Installation
### Prerequisites
Python 3.10 or above

### Dependencies
Flask

pip install Flask

### Run program

1. Install python 3.10 or above (3.12 used here)
2. pip install Flask
3. flask --app app run init-db
4. flask --app app run --debug 
5.  Follow the web link to see the website

### Additional 
The program utilizes Jinja for templates
